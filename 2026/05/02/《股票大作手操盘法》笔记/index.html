《股票大作手操盘法》笔记

一、世界观与基本信条

1.1 对投机的态度——应当以什么姿态参与

1.1.1 投机是一门严肃的生意，不是赌博

“无论是谁，如果天性具有投机倾向，就应当把投机视为一门严肃的生意，并诚心敬业。”
“所有参与此项事业的同行都应该下决心认真学习，尽己所能地充分发掘现有的数据资料，把自己从这项事业中领悟到的思想和观念作为个人最高的精神指导。”

1.1.2 “一分耕耘，一分收获”——必须亲自做行情记录、动脑思考、得出自己的结论

“你获得的成功将与你在自身努力中所表现出的诚意直接成正比。这种努力包括自己坚持做行情记录，自己动脑筋思考，并得出自己的结论。”
“你一定要亲自做行情记录……在亲力亲为的过程中，你会惊讶地发现，自己竟然也能做到灵感如泉涌。”

1.1.3 树立合理预期，不妄想暴富

“如果商人新开一家店铺，那么他不会指望第一年就从这家店铺获得25%以上的利润。但是，对于进行股票投机交易的人来说，25%的回报啥都不是。”

1.2 对市场的认知——市场是怎样的

1.2.1 市场永远不会错，观点可以千错万错

“观点可能千错万错，但市场永远不会错。”

1.2.2 从市场角度解读信息，而非自己的主观倾向

“在公布一则新闻后，你必须站在市场的角度，独立地分析它可能对行情造成的影响。你要尽力预测这则消息在一般投资大众心目中的心理效应。”

二、情绪纪律

2.1 戒痴——了解自己

2.1.1 每次都想赢是投机者头号公敌

“每一次轮流下注时，我们都想赢。这个弱点在某种程度上几乎成了投资者和投机者的头号公敌。”

2.1.2 希望与恐惧不能搅入交易——该怕的时候不要贪，该贪的时候不要怕

“一旦你把希望和恐惧这两种情绪搅进投机事业，就会面临一个极可怕的危险——你会被这两种情绪搅得几乎窒息，从而颠倒了它们的位置——在本该诚惶诚恐的时候满怀希望，在本该满怀希望的时候却诚惶诚恐。”

2.2 戒贪——切忌急躁

2.2.1 防止过度交易——不要天天投机

“假如你不放过每一个交易日，天天投机，那么你就不可能取得成功。每年可能只有寥寥可数的机会，才能允许自己下场开立头寸。除此之外的所有时间里，你都应该在局外旁观。”

2.2.2 快进快出是陷阱，永远抓不住大行情

“那些喜欢在当日之内的小规模市场波动中快进快出的投机客，永远不能在下一轮重大行情发生时捕捉到机会。”

2.2.3 不要急于将盈利转换为现金，放心让利润驰骋

“不要担心失去那些你从来没有真正拥有过的东西。要克制急于把账面利润转换成真正的现金的冲动。”

2.3 戒嗔——冷静面对失败

2.3.1 不因判断失误而暴怒自责，承认错误才能从中盈利

“我仅仅因为自己判断失误，就纵容自己暴怒和自责。”
“坦白承认错误，尽可能从中吸取教训，下次从中盈利。”

三、风险与资金管理

3.1 入场前——预设安全边界

3.1.1 永远在财务安全的前提下交易

“永远别做任何交易，除非你确知这样做在财务上是安全的，不会有任何经济问题。”

3.1.2 提前想清楚：万一判断错误，最多愿意承担多大风险

“万一你判断错误，你最多愿意承担多大的风险。”

3.1.3 拒绝没有依据的投机——每次出手要能自圆其说

“其中最失败的一种亏钱方式就是经不住推敲的投机。”

3.2 持仓中——控制损失

3.2.1 亏损不会自动了结，必须对首次小额亏损采取止损措施

“利润总是能够自己照顾自己，亏损却永远不会自动了结。投机者必须对第一次小额亏损采取止损措施，以此来确保自己不会蒙受巨额损失。”

3.2.2 绝不让亏损大到足以威胁未来操作

“确保投机事业持续进行的唯一出路是：小心守护自己的资本账户，绝不让亏损大到足以威胁未来操作的地步。留得青山在，不怕没柴烧。”

3.2.3 第一笔交易已亏损，绝不再继续跟进

“如果你的第一笔交易已然处于亏损状态，那么就绝不要再继续跟进，否则你会为你的执迷不悟付出代价。”

3.2.4 追加保证金通知＝立即平仓，不把”好钱”追加到”坏钱”里

“我从经纪商那里只得到过一种铁定无疑的’内幕’消息——追加保证金的通知。当这样的通知到达时，你应该立即平仓。这说明你站到了市场错误的一边。为什么要把’好钱’追加到’坏钱’里去？”

3.2.5 警惕”价值幻觉”——持仓中随机应变，关注市场的态度

“这些投资者当初买进股票的时候，相信这些股票的投资价值非常好。然而不幸的是，随着时间的推移，这些股票的基本面后来都遭遇了剧烈波动。自此以后，这些投资者所认为的’价值型股票’往往会转化成纯粹的投机型股票，甚至其中的一些股票已经不复存在了。”
“这些投资者才是真正的大赌徒。他们拍下赌注，就一条路走到黑。如果赌错了，他们就会输得什么都不剩。”

3.3 出场后——固化利润

3.3.1 成功平仓后取出一半利润储存起来

“每当他把一个成功的交易平仓了结后，都应该取出一半的利润储存到保险箱里积蓄起来。”

3.3.2 把钱握在手中，可以减少任性投机决策

“把钱握在手中的占有感，可以稍稍减轻你做出任性投机决策的冲动，而任性的投机决策会导致盈利流失。”

四、具体操作原则

4.1 耐心等待

4.1.1 等市场证实后再行动，不抢跑

“千万不要轻信自己的看法，一定要等到市场变化本身证实你的观点是正确的之后你再在自己的判断上签字画押。”
“如果你对某只或某些股票有明确的看法，千万不要急着一头扎进去，而是要从市场出发，耐心观察股票的行情走势，从中找到下一步行动的依据，然后伺机而动。”
“要等这只股票活跃起来并创新高后你再行动。因为只有到了这个时候，你的想法才被市场证实是正确的。”

4.1.2 耐心等待关键点，谨慎选择时机

“只要能够耐心等待市场到达关键点后再动手，就总能从交易中获利。”
“谨慎选择时机是绝对必要的，欲速则不达，操之过急的代价会非常惨重。”

4.2 独立研究

4.2.1 集中研究领头羊，不四面出击

“集中注意力研究当日行情最突出的那些股票，即’领头羊’。如果你不能从活跃的’领头羊’股票中获得利润，那么也就无法在整个股票市场中获得利润。”
“追随’领头羊’，但要保持思想的灵活性，因为股票市场也会不断抛弃旧的’领头羊’，产生新的’领头羊’。”

4.2.2 了解股票个性，不把单只视为整体趋势

“只要你肯下功夫，那么总有一天你会了解并尊重各类股票的个性，从而根据具体条件，去预测股票的运行趋势。”
“不把单只股票的价格变动情况看作这一类股票价格趋势变化的标志。”

4.2.3 不定义牛市/熊市——用上升趋势/下降趋势做判断

“我宁愿使用’上升趋势’和’下降趋势’这两个词，因为它们恰如其分地表达了市场在一定时间内即将发生的情形。”

4.2.4 不要等待事后的解释——市场行为本身才是实时的信号

“一只股票之所以会出现出人意料的变动趋势，总有它的理由。而且往往直到未来某个时间之后，你才能了解这个理由。”

4.2.5 内幕消息有局限：能告诉你何时买，不能告诉你何时卖

“你的这位掌握了公司内幕消息的企业家朋友，很容易告诉你何时应当买进。但是，他不可能也不愿意告诉你何时应当卖出。”

4.2.6 拿个小笔记本记下想法和市场信息，将来用得上

“你最好随身携带一个小笔记本，简略地记下一些有趣的市场信息：你的想法，也许将来用得上。”

4.3 出击与持有

4.3.1 警惕锚定思维：不因价格高而卖，不因价格跌而买

“绝不要因为某只股票看起来价格过高，就把它卖出去……恰恰应该研究判断，有没有任何因素可能阻止这只股票继续上涨。”
“绝不要因为某只股票从前一个最高点大幅下滑就把它买进来。这很有可能是一轮基于某种很充分的理由才形成的大幅下跌趋势。”

4.3.2 行情开始后及早入市，建立利润储备

“如果我没有在行情开始后不久就入市，就从来不会从这一轮行情中获得太大的收益。原因可能是，如果我没有及时入市，就丧失了一大部分利润储备。”

4.3.3 真正利润来自一开始就盈利的头寸

“我的经验向我证明，真正从投机交易中得来的利润，都来自那些一开始就盈利的头寸。”
“如果你正确地把握了这轮行情的时机，那么你投入的第一笔头寸应该从一开始就处于盈利状态。”

4.3.4 重视危险信号：冲高回落与模式失效

“一旦行情发生了不正常的变化，就是市场在向你闪烁危险信号，你绝对不能忽视这样的提醒。”
“如果该股票在穿越关键点之后没有按照应有的模式运动，那么这就是一个必须要密切注视的危险信号。”

4.3.5 大行情需要时间走完——给它时间展开，坚持到最后48小时

“真正的行情不会在一天之内就从开始走到结束，货真价实的行情总需要花上一些时日才能走到它的终点。”
“没有哪个重大的市场行情能在一天或一周时间内一蹴而就。它需要一定的时间才能逐步走完发生、发展、终结的全过程。在一轮行情中，大部分市场行情发生在整个过程的最后48小时内，因此这是持有头寸的最重要的时期。”
